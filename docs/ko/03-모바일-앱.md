# 3. TasMind 모바일 앱 (Android) 사용법

언제 어디서나 스마트폰으로 환경을 제어하세요.

---

## 3.1 실시간 기기 제어
- 앱 로그인 시 최신 기기 목록이 카드 형태로 표시됩니다.
- 카드를 탭하면 즉시 해당 기기가 제어됩니다.

## 3.2 FCM 푸시 알림
- 기기 오프라인 감지, 임계치 경보, 자동화 규칙 실행 시 스마트폰 상단바에 즉시 알림이 발행됩니다.

## 3.3 네트워크 오류 처리
- 서버 점검이나 네트워크 이탈 시, 모든 화면에서 통합된 팝업으로 오류를 안내합니다 (영어 팝업).

## 3.4 회원탈퇴
- 설정(Settings) 메뉴 하단의 `Delete Account`를 통해 앱에서도 바로 계정을 삭제할 수 있습니다.
