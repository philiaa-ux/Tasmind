# 5. 자주 묻는 질문 (FAQ) & 용어 사전

---

## ❓ FAQ

**Q1. Tasmota 기기가 대시보드에 안 떠요**  
A1. MQTT의 User=Topic 일치 여부, Port 8883 확인, Wi-Fi 연결 상태를 점검하세요.

**Q2. 기기가 오프라인으로 표시돼요**  
A2. 전원 확인, 공유기 재부팅, 기기와 공유기 간 거리를 좁혀보세요.

**Q3. 자동화 규칙이 작동 안 해요**  
A3. 규칙이 활성화(ON) 상태인지, 센서 값이 정상 보고되는지 확인하세요.

**Q4. AI 에이전트가 한국어로 대답 안 해요**  
A4. Settings에서 언어를 한국어로 설정하고 `/clear`로 대화를 초기화해보세요.

**Q5. 회원탈퇴하면 데이터는?**  
A5. 모든 기기 데이터, 규칙, 설정, 대화 기록이 영구 삭제되며 복구 불가능합니다.

**Q6. TasMind는 스마트홈 외에 다른 분야에도 쓸 수 있나요?**  
A6. 네! TasMind는 스마트팜(온실 환경 제어), 스마트빌딩(조명/냉난방 통합), 스마트팩토리(설비 모니터링) 등 다양한 산업 환경에 적용 가능한 범용 AIoT 플랫폼입니다.

## 📚 용어 사전

- **AP 모드**: 기기가 자체 Wi-Fi를 만들어 초기 설정 시 접속하는 모드
- **Friendly Name**: 사용자에게 보여지는 기기의 별칭 (예: "거실 조명")
- **GPIO**: 기기의 물리적 입력/출력 핀
- **MQTT**: IoT 기기 간 경량 메시징 프로토콜
- **RSSI**: Wi-Fi 신호 세기 (0에 가까울수록 좋음)
- **Self-Critique**: AI가 스스로 응답을 검토하여 위협을 차단하는 시스템
- **씬(Scene)**: 여러 기기 상태를 한 번에 변경하는 매크로
- **TLS**: 인터넷 통신을 암호화하는 보안 프로토콜 (8883 포트 사용)
- **토큰**: AI가 텍스트를 처리하는 기본 단위 (한글 1글자 ≈ 2-3토큰)
- **AI 에이전트**: 사용자 명령을 이해하고 기기를 제어하는 지능형 비서 시스템
