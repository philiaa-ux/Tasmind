# 1. Tasmota 기기 초기 설정 (TLS 8883 & MQTT 보안 강화)

스마트 플러그, 릴레이 보드, 센서 노드 등의 IoT 장치에 Tasmota를 적용하고 TasMind 보안 서버망에 안전하게 연결하는 과정입니다.

---

## 1.1 펌웨어 설치 (누구나 할 수 있어요!)

> 🛠️ **필요한 것**: Tasmota를 설치할 스마트 기기, USB 케이블, Chrome 또는 Edge 브라우저

1. **웹 설치 프로그램 접속**  
   Chrome이나 Edge 브라우저를 열고 [Tasmota Web Installer](https://tasmota.github.io/install/)로 이동하세요.
2. **설치 메뉴 설명**  
   Tasmota(english) : 펌웨어 종류 선택(각 나라 언어별, 장치별, 개발 버전 선택가능). 기본은 Tasmota(english).  
   버전 선택(날자로 표시) : 펌웨어 개발 및 수정 버전 및 변경 사항 표시, 기본이 가장 최신 버전.  
   Filter by date: 펌웨어 버전을 드롭 다운 메뉴로 선택 및 조회가 가능.  
   Flash Speed: 펌웨어 다운 및 설치 속도 선택, 2000000이 기본.
   ** Upload factory firmware with the file dialog or by dragging and dropping onto the dashed region 공간은 파일 형태의 수정한 펌웨어나 공장 초기화 펌웨어 파일 업로드할 수 있다. 파일 형태의 펌웨어를 업로드 가능
   

4. **기기 연결 및 설치**  
   USB 케이블로 기기를 PC에 연결하고 화면의 **"connect"** 버튼을 클릭하면 자동으로 설치가 진행됩니다 (약 2~3분 소요).  
   connect 버튼을 누르면 팝업 창이 나타남. usb에 연결된 목록중에서 tasmota 펌웨어를 설치할 장치를 선택하고 connect버튼을 누르면 설치화면화면이 나타난다.  
   ** 본인의 esp32 장치나 모듈이 나타나지 않으면 usb포트로 인식할 수 있게 드라이버(CP210x, CH34x (CH340, CH341 등))을 검색하여 본인 pc에 다운로드 한후 설치해야 한다.  
   기존에 있는 펌웨어 삭제 및 설치가 완료가 되면 wifi연결 설정 화면에서 장치를 연결할 공유기의 wifi를 선택하고 비빌번호를 입력하면 wifi에 접속된다.  
   wifi 접속이후에 설치 메뉴에서 visit device 버튼을 클릭하면 해당 장치의 tasmota 펌웨어 웹화면이 나타난다. 이때 브라우저의 주소창을 보면 192.168.0.xxx등의 주소가 표시되는데, 이것이 펌웨어를 설치한 장치의 ip주소이다.
   앞으로 브라우저의 주소창에 해당 ip 주소만 입력하면 tasmota 웹화면으로 접속하여 설정이나, 작동 등을 직접 할 수 있다.
   그리고 장치의 ip를 잊었을 경우에는 이전과 같이 connect를 한후에 이미 tasmota가 설치되어 있는 경우는 설정 메뉴로 자동 이동된다. 이전과 같이 visit device를 누르면 알 수 있다. 그리고 여기서 wifi변경도 가능하다.

## 1.2 Wi-Fi 연결하기 (AP 모드, tasmota 설치과정에서 wifi 설정을 못하였을 경우)

1. 기기 부팅 후 Wi-Fi 목록에서 `tasmota-xxxxxx-xxxx` 네트워크에 접속하세요.
2. 브라우저에서 `http://192.168.4.1`로 접속합니다.
3. 집 Wi-Fi 이름(SSID)과 비밀번호를 입력하고 저장(Save)하면 기기가 재부팅되며 자신의 공유기에 연결됩니다.

---

## 1.3 모듈 설정 (GPIO 핀 할당)

1. 기기가 할당받은 IP 주소로 접속 (`http://192.168.0.xxx`)
2. **Configuration → Configure Module**로 이동합니다.
3. 기기 용도에 맞게 GPIO 핀을 설정하세요.
   - 릴레이(스위치): `Relay 1`, `Relay 2`
   - 온습도 센서: `AM2301` (DHT22)
   - 조도/가스 센서: `ANALOG`
4. 저장(Save) 후 재부팅합니다.

---

## 1.4 MQTT 보안 설정 (가장 중요!)

> ⚠️ **User와 Topic은 반드시 동일해야 합니다!**

1. **Configuration → Configure MQTT**로 이동합니다.
2. 아래 표에 따라 정확히 입력하세요.

| 설정 항목 | 입력값 | 설명 |
|-----------|--------|------|
| **Host** | TasMind MQTT 서버 주소 | 관리자에게 받은 주소 입력 |
| **Port** | `8883` | **반드시 8883 사용!** (TLS 보안 포트) |
| **User** | `tasmota_ai2` (예시) | 장치 고유 식별자 |
| **Topic** | `tasmota_ai2` (예시) | **User와 동일하게!** |
| **Password** | 발급받은 토큰 | 장치별 고유 비밀번호 |

- ✅ User = Topic → 연결 성공!
- ❌ User ≠ Topic → 연결 실패 (보안 인증 오류)

---

## 1.5 기기 이름 설정 (Friendly Name)

1. **Configuration → Other**로 이동합니다.
2. device name : 장치를 쉽게 이해하기 위한 이름(예시: 1층장치, 하우스1 등)으로 변경합니다. 
3. **Friendly Name 1~N**에 릴레이나 플러그 등과 같이 on/off되는 출력 장치 이름을 한글로 직관적인 이름을 입력하세요.  
   (예: `거실 온풍기`, `침실 메인조명`)
4. `MQTT Enable`이 체크되었는지 확인하고 저장합니다.
5. friendly name과 device name은 장치를 쉽게 이해할 수 있는 별명입니다. 설정이 완료되면 Tasmind에서 자동으로 별명으로 매칭됩니다.
