# 4. AI 에이전트 (AI Chat) 및 보안 방어

웹 대시보드 하단 플로팅 패널 및 모바일 앱 탭에서 AI와 대화할 수 있습니다.

---

## 4.1 퀵 액션 칩 (빠른 명령어)
- 고정 칩: `🧹 Reset Chat` (대화 초기화), `📋 Device List` (기기 목록), `📊 Device Status` (상태 확인)
- 동적 칩: 자주 사용하는 커스텀 명령어 3개가 자동으로 표시됩니다.

## 4.2 대화 초기화 (`/clear`)
- `🧹 Reset Chat` 칩 클릭 또는 `/clear` 입력 시 채팅 내용이 삭제되고 AI의 대화 컨텍스트가 완전히 초기화됩니다.

## 4.3 Self-Critique 보안 방어막
- 시스템 프롬프트 요청, 악성 OTA 펌웨어 변경 시도, 모든 장치 동시 OFF 명령 등 위협이 감지되면 AI가 자체 차단합니다.
- 공격 시도는 감시 로그에 자동 기록됩니다.
