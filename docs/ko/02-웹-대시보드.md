# 2. TasMind 웹 대시보드 완전 정복

로그인 후 만나는 4가지 주요 화면(Home, Devices, Rules, Settings)을 자세히 알아봅니다.

---

## 2.1 Home (대시보드 홈)

### 🎛️ 실시간 기기 제어 카드
- 각 기기가 카드 형태로 표시되며, 버튼 클릭 시 1초 내에 실제 릴레이가 ON/OFF 됩니다.
- **상태 표시**: 🟢 Online (정상) / 🔴 Offline (연결 끊김)

### 📊 실시간 센서 데이터
- 온도, 습도, 대기질, pH 등 센서 값이 실시간 숫자로 표시됩니다.

### 📈 시계열 센서 차트
- 드롭다운으로 기기와 센서 타입을 선택하면 트렌드 그래프를 볼 수 있습니다.
- **기간 필터**: `1H` (1시간), `6H` (6시간), `24H` (하루), `7D` (7일)  
  *7D는 1시간 단위 평균값으로 주간 패턴을 분석합니다.*

### 👤 개인설정 위젯
- **Current Tier**: 현재 등급 (Tier 1~3) 표시
- **Request Upgrade**: 등급 업그레이드 요청 버튼
- **Daily Token Usage**: 오늘 사용한 AI 토큰량을 프로그레스 바로 표시
- **Membership Withdrawal**: 계정 영구 삭제 (모든 데이터 소멸)

---

## 2.2 Devices (장치 목록 관리)

- **네트워크 정보**: IP 주소, Wi-Fi 신호 강도(RSSI), 마지막 통신 시간, 가동 시간(Uptime) 표시
- **명칭 편집**: 기기 이름을 더블클릭하여 즉시 수정 가능
- **기기 제거**: 사용하지 않는 기기를 계정에서 영구 삭제

---

## 2.3 Rules (자동화 규칙)

### 🔧 조건부 규칙 (IF-THEN)
- 예: *"온도가 30도 초과 시 에어컨 켜기"*
- 트리거(조건)와 액션(실행)을 시각적 양식으로 등록하면 백엔드에서 즉시 적용됩니다.

### 🔄 자동화 활성화 스위치
- 각 규칙 우측의 토글로 규칙을 켜거나 끌 수 있습니다 (삭제 없이 일시 중지 가능).

### 🎬 씬(Scene) 관리
- **영화 모드**, **외출 모드**, **취침 모드** 등 여러 기기를 한 번에 제어하는 매크로를 설정할 수 있습니다.

---

## 2.4 Settings (개인 설정)

- **계정 프로필**: Google 연동 정보 (닉네임, 이메일)
- **다국어 선택**: 🇰🇷 한국어 / 🇺🇸 English / 🇯🇵 日本語 (클릭 한 번으로 전환)
- **타임존 설정**: 기기가 위치한 국가/지역 시간대를 지정하여 데이터 차트 기준 시간을 조정합니다.
