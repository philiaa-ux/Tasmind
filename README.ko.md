# TasMind (태스마인드)

**TasMind**는 AI 기반의 통합 IoT 플랫폼으로, 스마트홈을 넘어 **스마트팜, 스마트빌딩, 스마트팩토리** 등 다양한 환경에서 지능형 기기 제어와 자동화를 제공합니다. 하나의 AI 에이전트가 모든 것을 연결합니다.

> 🚀 하나의 플랫폼, 무한한 가능성. AI로 어떤 환경이든 제어하세요.

---

## ✨ 주요 기능

- 🔌 **Tasmota 연동** – TLS 보안 MQTT (8883 포트) 지원
- 📊 **웹 대시보드** – 실시간 기기 제어, 센서 차트, 자동화 규칙
- 📱 **모바일 앱** – 원격 제어 및 FCM 푸시 알림
- 🤖 **AI 에이전트 (채팅)** – 자연어로 복합 명령, 센서 데이터 분석, 자동화 시나리오 생성 및 적용, Tasmota 고급 설정까지 한 번에 처리
- 🧠 **자동화 및 시나리오** – 시스템 규칙(서버 기반)과 물리 규칙(Tasmota Rule) 모두 지원, 가상 노드를 통한 다중 장치 연동, 커뮤니티 시나리오 검색 및 적용
- 🌍 **다양한 도메인 지원** – 주거, 농업, 건물, 산업 현장까지 확장 가능

## 🧠 AI 채팅으로 더 스마트하게

TasMind의 AI 채팅은 단순한 질의응답을 넘어, 다음과 같은 강력한 기능을 제공합니다.

| 기능 | 설명 | 예시 명령 |
|------|------|-----------|
| **복합 명령 처리** | 여러 장치를 한 문장으로 제어 | *"거실등 켜고, 에어컨 26도로 설정하고, 5분 후 거실등 꺼줘"* |
| **센서 데이터 분석** | 실시간 조회 및 24시간/주간 트렌드 요약 | *"안방센서의 지난 24시간 온도 변화 분석해줘"* |
| **자동화 시나리오 생성** | 자연어로 조건/동작을 설명하면 AI가 DSL로 변환하여 등록 | *"온도 28도 이상이고 습도 60% 이하면 에어컨 켜는 시나리오 만들어줘"* |
| **규칙 유형 자동 선택** | 시스템 규칙(서버)과 물리 규칙(Tasmota Rule) 중 최적 추천 | *"이 규칙을 물리 규칙으로 변환해줘"* |
| **Tasmota 고급 설정** | TelePeriod, Interlock, 셔터/블라인드, 백로그 명령 등 | *"하우스1의 Telemetry 주기를 5분으로 변경해줘"* |
| **유령 노드 정리** | 오래된 미통신 장치 일괄 삭제 | *"10분 이상 통신 끊긴 장치 삭제해줘"* |

> 💡 **대시보드 vs AI 채팅**: 단순 ON/OFF는 대시보드가 빠르지만, 복잡한 작업이나 여러 명령을 한 번에 처리할 때는 **AI 채팅이 훨씬 효율적**입니다. 토큰 사용량도 최적화되어 있어 경제적입니다.

## 🛠️ 빠른 시작

1. Tasmota 펌웨어를 기기에 플래싱하세요.
2. MQTT 설정에서 **Port 8883**과 **User = Topic** 일치를 확인하세요.
3. TasMind 대시보드에 접속하여 자동화를 시작하세요.
4. AI 채팅에 *"내 장치 목록 보여줘"* 라고 물어보고, 원하는 자동화를 자연어로 설명하세요.
