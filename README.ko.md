# TasMind (태스마인드)

**TasMind**는 AI 기반의 통합 IoT 플랫폼으로, 스마트홈을 넘어 **스마트팜, 스마트빌딩, 스마트팩토리** 등 다양한 환경에서 지능형 기기 제어와 자동화를 제공합니다. 하나의 AI 에이전트가 모든 것을 연결합니다.

> 🚀 하나의 플랫폼, 무한한 가능성. AI로 어떤 환경이든 제어하세요.

## ✨ 주요 기능
- 🔌 **Tasmota 연동** – TLS 보안 MQTT (8883 포트) 지원
- 📊 **웹 대시보드** – 실시간 기기 제어, 센서 차트, 자동화 규칙
- 📱 **모바일 앱** – 원격 제어 및 FCM 푸시 알림
- 🤖 **AI 에이전트** – 자연어 제어 및 자체 보안 방어 시스템
- 🌍 **다양한 도메인 지원** – 주거, 농업, 건물, 산업 현장까지 확장 가능

## 🛠️ 빠른 시작
1. Tasmota 펌웨어를 기기에 플래싱하세요.
2. MQTT 설정에서 **Port 8883**과 **User = Topic** 일치를 확인하세요.
3. TasMind 대시보드에 접속하여 자동화를 시작하세요.
